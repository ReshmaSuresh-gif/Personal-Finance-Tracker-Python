# Personal Finance Tracker - Python 💰

## Project Overview

The Personal Finance Tracker is a Python mini project designed to help users manage their personal finances efficiently. This application allows users to track income, expenses, savings, and monitor their financial activities in a simple and organized way.

This project was developed to improve Python programming skills and understand concepts such as loops, conditional statements, functions, file handling, and data management.

## Features

* Add income details
* Track expenses
* View balance/savings
* Display financial summary
* Simple menu-driven interface
* User-friendly system

## Technologies Used

* Python

## Python Concepts Used

* Variables
* Conditional Statements (`if-else`)
* Loops (`while`)
* Functions
* Lists/Dictionaries
* User Input Handling

## How to Run the Project

1. Install Python on your system.
2. Download or clone this repository.
3. Open terminal or command prompt.
4. Run the following command:

```bash id="w2d8lq"
python finance_tracker.py
```
